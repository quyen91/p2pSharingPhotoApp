# p2pSharingPhotoApp
### Name App
HotPhoto
### Main function
- Upload photo to share all connect in network
- View all photo shared on same wifi network
- Like, vote photo

### Require Technique
- Wifi connect using p2p
- Connect and use SQlite to save data
- Using layout properly
- Multithreading
